SummaryÂ¶
















We've created three models each iteratively adding more features to test for increased performance. In line with expectations our advanced model with 18 features reached a RMSE clocking in at better than 156831 compared with almost twice as much for the first model.Â¶
The 18 features model may underperform if being used on a test dataset that has houses filled with only duplexes which lack many of the features Bill Gates' house might have. This shows that it's always important to carefully, and in many cases manually evaluate feature selection for our machine learning models to suit the situation.Â¶
